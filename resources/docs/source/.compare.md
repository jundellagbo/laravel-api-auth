## Register User

User registration API endpoint

> Example request:

```bash
curl -X POST \
    "http://localhost/api/v1/auth/register" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Authorization: Bearer {token}" \
    -d '{"firstName":"illo","lastName":"enim","email":"aperiam","password":"consequuntur","gender":"laborum","birthDate":"dolor","watsappNumber":"illum","phoneNumber":"consequatur","passportNumber":"suscipit","visaNumber":"magni","accessCardNumber":"magnam"}'

```

```javascript
const url = new URL(
    "http://localhost/api/v1/auth/register"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer {token}",
};

let body = {
    "firstName": "illo",
    "lastName": "enim",
    "email": "aperiam",
    "password": "consequuntur",
    "gender": "laborum",
    "birthDate": "dolor",
    "watsappNumber": "illum",
    "phoneNumber": "consequatur",
    "passportNumber": "suscipit",
    "visaNumber": "magni",
    "accessCardNumber": "magnam"
}

fetch(url, {
    method: "POST",
    headers: headers,
    body: body
})
    .then(response => response.json())
    .then(json => console.log(json));
```



### HTTP Request
`POST api/v1/auth/register`

#### Body Parameters
Parameter | Type | Status | Description
--------- | ------- | ------- | ------- | -----------
    `firstName` | string |  required  | max:100
        `lastName` | string |  required  | max:100
        `email` | string |  required  | max:100,unique
        `password` | string |  required  | max:20,min:5,unique,confirmed:password_confirmation
        `gender` | string |  optional  | max:6
        `birthDate` | string |  optional  | MM-DD-YYYY
        `watsappNumber` | string |  optional  | max:15
        `phoneNumber` | string |  optional  | max:15
        `passportNumber` | string |  optional  | max:25
        `visaNumber` | string |  optional  | max:25
        `accessCardNumber` | string |  optional  | max:25
